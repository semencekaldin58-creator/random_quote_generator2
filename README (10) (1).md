## Описание программы

`Random Quote Generator` — это приложение с графическим интерфейсом (GUI) на Python, которое позволяет получать порцию вдохновения или мудрости одним кликом. Программа хранит базу известных цитат и случайным образом выводит их на экран. Вы также можете пополнять базу собственными любимыми цитатами, привязывая их к конкретным авторам и темам.

### Основные возможности:

1.  **Генератор цитат:** Случайный выбор цитаты из встроенного или добавленного вами пула.
2.  **Пользовательская база:** Вы можете добавлять свои цитаты, указывая текст, автора и тему. Программа защищена от добавления пустых строк.
3.  **Фильтрация (2 уровня):** Возможность фильтровать генерируемые цитаты и историю как по автору, так и по теме одновременно.
4.  **Сохранение истории:** Все сгенерированные вами цитаты, а также добавленные новые фразы сохраняются в JSON-файлы (`quotes_history.json` и `quotes_pool.json`) и остаются доступны после перезапуска.

## Инструкция по запуску

Приложение использует только стандартные библиотеки Python, сторонние зависимости не требуются.

1.  Убедитесь, что у вас установлен **Python 3**.
2.  Скачайте файлы проекта в одну папку.
3.  Откройте терминал (командную строку) в этой папке.
4.  Запустите программу командой:
    ```bash
    python random_quote_generator.py
    ```

## Примеры использования (Тесты)

*   **Тест 1: Генерация базовой цитаты.**
    Нажмите кнопку "Сгенерировать цитату". В текстовом поле посередине экрана появится случайная цитата, а в таблице снизу — запись о её генерации.
*   **Тест 2: Добавление новой цитаты.**
    В верхнем блоке заполните поля: Текст - "Знание — сила", Автор - "Фрэнсис Бэкон", Тема - "Знания". Нажмите "Добавить". Появится сообщение об успехе. Теперь эта цитата в пуле.
*   **Тест 3: Проверка фильтров и генерации.**
    Выберите в поле "Фильтр по теме" значение "Знания" (которое вы только что добавили). Нажмите "Сгенерировать цитату". Программа выдаст именно цитату Бэкона. Таблица истории также покажет только те цитаты, которые соответствуют этой теме.
*   **Тест 4: Защита от пустого ввода.**
    Оставьте поле "Автор" пустым и попытайтесь добавить цитату. Программа покажет ошибку "Все поля (Текст, Автор, Тема) должны быть заполнены!".
*   **Тест 5: Сохранение данных.**
    Закройте приложение и запустите его снова. Ваша добавленная цитата Бэкона останется доступна в фильтрах, а в таблице снизу загрузится вся предыдущая история генераций.
